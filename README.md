<img width="1358" alt="Screenshot 2025-02-23 at 13 29 54" src="https://github.com/user-attachments/assets/400c9a71-21ef-443a-adde-6b2341f4eb51" />![2025-02-24 4 32 52 PM](https://github.com/user-attachments/assets/b026c73e-7de3-4f44-af57-a9a7f169a65c)EventsManager is a web-based application for event search, ticket purchase, and management.

, Functionality

View a list of upcoming film premieres, theater performances, and sporting events.

, Registration and login of users.

, The ability to edit a profile.

🎟️ Purchase of tickets for events.

, Leaving reviews and comments.

, A page for each ticket with a detailed description of the event.

🛠️ Technology

React.js - a library for building the interface.

Context API and Redux - application state management.

JavaScript is the main development language.

CSS and HTML - page styling and structure.

Jest - functional testing.

Testing API - checking API requests.

JSON-Server is an emulation of a server database.
